# AI Self-Improvement Recommendation Engine

## Overview
An AI-powered system that recommends personalized learning content based on user goals, level, and behavior.

## Features
- Semantic search using embeddings
- Personalized ranking
- Machine learning-based scoring
- Feedback loop for continuous improvement
- Streamlit UI

## Tech Stack
- Python
- scikit-learn
- sentence-transformers
- Streamlit
- SQLite

## How to Run
pip install -r requirements.txt
streamlit run app.py
